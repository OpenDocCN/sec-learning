## 下载

### Docker

```
docker pull apachecn0/sec-learning
docker run -tid -p <port>:80 apachecn0/sec-learning
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install sec-learning
sec-learning <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g sec-learning
sec-learning <port>
# 访问 http://localhost:{port} 查看文档
```
