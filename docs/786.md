## 介绍说明

Fofa Pro View 插件告诉您网站的托管位置（国家、城市）、IP 的所有者以及打开的其他服务/端口。

用于 Chrome 的 Fofa Pro 视图插件会自动检查 fofapro 是否有当前网站的任何信息。网站是否也运行 FTP、DNS、SSH 或一些不寻常的服务，有了这个插件，你可以看到 Fofa Pro 在一个给定的网站/域名上收集的所有信息。
