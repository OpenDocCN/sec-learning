# User-Agent Switcher

> 原文：[http://book.iwonder.run/Tools/游览器插件/User-Agent Switcher.html](http://book.iwonder.run/Tools/游览器插件/User-Agent Switcher.html)

## 介绍说明

User-Agent Switcher 是一款可以模拟其他游览器访问页面的插件

目前支持

```
Chrome
Internet Explorer
iOS
Android
Windows Phone
Firefox
Opera
Safari
您自定义的用户代理字符 
```
